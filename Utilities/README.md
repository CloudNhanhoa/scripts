# Sript Utilities

## CMD_log (CentOS & Ubuntu)

## Create Swap
```sh 
curl -Lso- https://raw.githubusercontent.com/nhanhoadocs/scripts/master/Utilities/create_swap.sh | bash -s <size>
```

- Lưu ý: `<size>` chính là size swap (GB)
